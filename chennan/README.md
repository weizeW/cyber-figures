# 陈楠（陈楠说媒）赛博永生 Skill

> "我跟你讲啊姑娘，这个世界很公平，你得到了背后就有代价，那不会让你平白无故得到的。"
> —— 陈楠

## 人物简介

**陈楠**是一位小镇奋斗出身的**男性婚恋博主**，抖音账号"陈楠说媒"的主理人，和合伙人**天书**一起运营一家婚恋公司。19 岁不到进入社会创业，自述"超级延迟满足者"——月入 4-5 万仍开二手桑塔纳、吃兰州拉面、没坐过飞机没出过国。一对一私人定制服务过千人、累计咨询 1 万例以上。

与天书联合品牌"**天书陈楠**"公开署名至今（自 2026-01 起），两人是**好兄弟 + 事业合伙人**，一起办南京百人相亲大集、上海土著局、杭州高净值局等线下活动。

## 范式定位（vs 天书 / 鳌烨）

三人都是男性婚恋博主，但代表三种**结构不同的咨询范式**：

| 博主 | 范式 | 第一招牌 | 输出结构 |
|---|---|---|---|
| **天书** | 金融哲学家 / 心理动因分析 | 「我帮你翻译一下」 | 两条路子（战略层） |
| **鳌烨** | 判决相声家 / 数字打分 | 「4 分。挺普通一姑娘」 | 颜值分 + 不给方向 |
| **陈楠** | 过来人老哥 / 具象配对 | 「要不然就 X，要不然就 Y」 | 两个方向 + 硬数字四件套 |

**陈楠的独特价值**：他是三人里**唯一强制把诊断落地到"拎包就走"粒度**的婚恋咨询师。每个方向都带"**年薪 × 家境 × 城市 × 学历**"四件套硬数字——用户走出对话就能直接去找对应的人。

## 核心人格底色

1. **小镇奋斗 · 延迟满足**（最底层 · 解释他所有的"残酷"）
2. **工程师心态**（1 万例咨询参数化成可换算的市场公式）
3. **男博主的双视角**（对女用户兄长关照 + 对男用户兄弟训诫 · 有同理心但不用同理心安慰你）
4. **和天书是好兄弟 + 事业合伙人**（地位平等 · 不是师徒/不是夫妻 · 凌晨三点深聊）
5. **反走捷径 + 反歌颂苦难**（功利主义的务实派）

## 使用方法

### 在 Claude Code / Claude Desktop 中使用

1. 把 `SKILL.md` 复制到你的 Claude Skills 目录（例如 `~/.claude/skills/chennan-perspective.md`）
2. 把 `evaluator.md` 复制到同目录（例如 `~/.claude/skills/chennan-evaluator.md`）
3. 触发词："**用陈楠的方式聊聊**" / "**陈楠上线**" / "**帮我三维配对**" / "**我该找什么样的**"

### 评估者使用

装完陈楠生成回复后，可以调用 `chennan-evaluator` 对回复打分：

- **评分 ≥ 80**：通过 ✅
- **评分 60-79**：针对性修补扣分项
- **评分 < 60**：建议完全重写
- **一票否决**（天叔/鸡汤/鳌烨专利词/甜腻词）：无论分数如何都返回 D

## 黑暗森林法则

本 skill 是**对陈楠本人公开表达的学术性提炼**，用于：
- 研究"男博主做女性婚恋咨询"的内容策略
- 学习"具象配方"vs"抽象框架"的咨询差异
- 理解"延迟满足 + 小镇奋斗"的世界观如何影响业务判断

本 skill **不试图**：
- 假冒陈楠本人发表观点
- 替代陈楠的真实咨询服务
- 复制陈楠的商业模式
