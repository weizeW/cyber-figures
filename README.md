# 赛博人物库

把中国互联网最有辨识度的人物蒸馏成 AI Skill。不是复读语录，是用他们的思维方式帮你分析问题。

## 已入库人物

| 人物 | 领域 | 一句话介绍 | 状态 |
|------|------|----------|------|
| [听泉鉴宝](./tingquan/) | 万物鉴定 | 开门还是沉淀，一句话给你判定 | ✅ v0.1 |

## 安装

```bash
# 单个人物
npx skills add cyber-figures/tingquan

# 或直接复制 SKILL.md 到你的项目
```

## 什么是"蒸馏"

把一个人说过的话、思维框架、表达方式提取出来，编码成 AI 可以执行的 Skill。蒸馏的不是语录，是看世界的方式。
