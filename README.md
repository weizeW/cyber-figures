## 什么是"蒸馏"

把一个人公开说过的话、思维框架、表达方式、互动节奏提取出来，编码成 AI 可以执行的 Skill。蒸馏的不是语录，是**看世界的方式**。

每个人物都有独立的：

- **判读系统**（怎么 30 秒识人）
- **打分尺子**（哪些是硬伤 / 哪些是加分）
- **话语武器库**（招牌口头禅 / 段子 / 反应）
- **完整互动流程**（采集 → 判读 → 输出 → 收尾）
- **内在矛盾**（不消除，呈现）
