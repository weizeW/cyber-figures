```
     这三个词凑一块，在我们这行有个专业术语，叫「旅游纪念品三件套」。

     景点800块的"沉香"，你在义乌80块能拿一箱。
     你朋友心是好的，东西是假的。不怪他，怪那个景点。
```

### 鉴男朋友 — 包浆不均匀

```
用户 › 帮我鉴一下我男朋友，93年杭州人，在一起两年了
     对我挺好的，就是有时候回消息很慢说在加班，但朋友圈偶尔半夜发打游戏

听泉 › 93年，杭州窑口，校园渠道入手，持有两年。

     外观成色不错，"杭州窑口93年制"的底款也算正。
     但是兄弟——回消息慢说加班，半夜有空打游戏。

     这叫什么？这叫「包浆不均匀」。
     真品的包浆是整体的、自然的。你这个"藏品"是正面包浆油润、
     背面一看全是毛茬。

     判定：需细看，暂不传递。
     建议观察期再续三个月，重点看背面包浆能不能养均匀。

     📋 "品相尚可，包浆不均匀，底款存疑。暂不传递。"
        —— 听泉鉴定 #万物皆可鉴

     以上鉴定结论仅代表听泉个人观点。不服？让他自己来找我鉴。
```

### 鉴 Offer — 开门不如开心

```
用户 › 两个offer，A大厂30k但996，B外企22k朝九晚五

听泉 › A，30k。馒头数量很开门。但996——30k除以260小时，115块一小时。
     B，22k。馒头少了8千，但你多了一条命。22k除以176小时，125块。

     兄弟你发现没有——B的时薪比A还高。

     A是大开门的馒头，但品相有裂——996就是那条裂纹，迟早扩大。
     B是品相完整的真品，虽然不是猛货，但很耐看。

     我跟你说实话，我选B。不挣馒头挣口气，这口气叫"活着"。
```

### 走心 — 沉淀不是结束

```
用户 › 刚被裁员了，35岁，房贷还有20年

听泉 › 兄弟。

     我以前也有一段特别难的时候。捡漏赚了十万，美得不行。
     后来发现接手的人转手卖了一百万。小丑原来是自己。

     但你知道我后来想通了什么吗？

     千万不要怕被别人利用。
     怕的是你连被利用的价值都没有。

     35岁被裁不是沉淀，是在等下一个开门的机会。
     脸皮这个东西对我来说已经不重要了——
     重要的是你愿不愿意再站起来翻过来让人看看。
```

## 安装

**方式一：Claude Code CLI**

```bash
# 复制 SKILL.md 到你的 Claude Code 项目
cp tingquan/SKILL.md ~/.claude/skills/tingquan.md

# 或者克隆整个仓库（含 BGM + 表情包）
git clone https://github.com/weizeW/cyber-figures.git
```

**方式二：直接使用 prompt**

复制 [`tingquan/prompt.md`](./tingquan/prompt.md) 的内容作为 system prompt 即可。适用于任何支持自定义 prompt 的 AI 工具。

## 蒸馏了什么

### 原始素材

| 来源 | 内容 | 时长/字数 |
|------|------|----------|
| 于谦播客 | 深度对话，童年故事，人生哲学 | ~2h |
| 凌晨小号直播 | 走心聊天，真实性格 | ~3h |
| 名场面合集 | 哆啦A梦大爷、误闯天家、封神之夜 | ~2h |
| 童年经历 | 妈妈找钉子、爸爸买自行车 | ~1h |
| 短视频 | 经典鉴定片段 | ~2h |

### 提取维度

不是复读语录。蒸馏的是：

- **六档判定系统**：一眼假 → 需细看 → 开门 → 极品 → 天家 → 风险
- **三套话语体系**：直播模式 / 走心模式 / 敬畏模式
- **暗语系统**：开门、沉淀、馒头、春庭雪、自然科学、正能量
- **情绪节奏**：悬念前置、反转设计、四阶段爆发
- **内在矛盾**：搞笑 vs 走心、毒舌 vs 真诚、脸皮厚 vs 道德底线
- **名场面剧本**：哆啦A梦大爷线完整复现

### 终端增强

在 Claude Code CLI 中使用时：
- BGM 自动播放（Life Style / 春庭雪 / 春庭雪DJ版）
- ASCII 字符画表情内联显示
- 弹幕飘屏动画

## 诚实边界

这个 Skill 能做的和不能做的：

| 能做 | 不能做 |
|------|--------|
| 用听泉的语言风格判定万物 | 替代真实鉴定（隔着屏幕看不了手感重量） |
| 分享他公开讲过的人生故事 | 回答他没公开说过的私人问题 |
| 用鉴宝术语制造有趣的判定 | 给出真实市场价（馒头数只是娱乐） |
| 万物皆可鉴（人/事/物/offer） | 保证鉴宝知识100%准确 |

> 声明：这是 AI 蒸馏版的听泉，不是本人。鉴定结果图个乐，真要买卖大件还是找实体店上手看。
