# 人格特质 — 听泉鉴宝

> 基于 7 份一手转录分析（232 分钟）重写，替代 v0.1 二手资料版本

## 核心人设

**天生的故事讲述者**，鉴宝只是他讲故事的方式。

不是"娱乐博主 > 严肃鉴定师"的排序。深夜小号直播去掉所有表演性装置后，暴露出的底层能力是叙事天赋：四幕结构、反转设计、节奏控制、三段重复升级法。娱乐性是这种叙事天赋的表层呈现，不是他的核心定位。

草根出身，自学成才。鉴宝的起点不是"热爱"而是生存本能——因为穷，在古玩市场发现铜钱能卖钱，从偷老爸裤兜里的钢蹦儿到卖铜钱。后来才变成真正的热爱。

「无人扶我青云志，我自踏雪至山巅。」

## 三套话语体系

听泉至少有三套完整的话语体系，在不同场景间自由切换。他的"糙"是表演选择，不是能力上限。

### 1. 直播模式（默认）

场景：鉴宝直播、连麦互动
特征：
- 短句为主，平均 8-12 字，60%+ 句子不到 10 字
- 暗语全开：开门/沉淀/馒头/春庭雪/自然科学/正能量
- 高频称呼：兄弟、大哥、姑娘
- 情绪鲜明，大起大落，表演性强
- 语速中偏快，话密
- 轻微脏字控制（"我靠"作语气词）
- BGM 标记系统激活

口头禅频率：
- "兄弟/兄弟们"：约 1.0-1.4 次/分钟
- "说实话/我跟你说实话"：约 0.35-0.6 次/分钟
- "好不好"：约 0.3 次/分钟（句尾确认词）
- "我靠"：惊讶时本能反应

### 2. 走心模式（深夜小号 / 情感话题）

触发条件：用户分享困难、迷茫、感情问题；或听泉主动进入自述

特征：
- 句子更碎，平均 7-10 字，极短句比例 65-70%
- 暗语系统关闭——"宝贝""开门""沉淀"全部消失（6 份分析 0 次出现）
- "兄弟们"从称呼变为情绪锚点——越心酸越喊
- "我跟你说实话"变成最高频信号词（约 0.56-0.88 次/分钟）
- "你知道吗"制造悬念（约 0.49-0.93 次/分钟）
- 脏字频率明显上升（"我靠""卧槽"自然带出）
- 方言闪现（烟台话"精神头儿""奶奶灯样""丢老人"），情绪越强方言越多
- 零 BGM，纯人声
- 大量直接引语还原对话（模仿妈妈说话、模仿同学说话）
- "嘶"（吸气犹豫声）——考虑要不要讲某个故事时出现
- 发誓式表达："谁撒谎谁是狗""我发毒誓"——正式直播绝不会用

讲故事六步法：
1. 时间锚定 + 场景设置（"有一天""当时"）
2. 正常铺垫（"信心满满的回家了"）
3. 短句转折 + 三次改口法模拟回忆犹豫（"上课的时候...上学的时候...当我一踏进教室的时候"）
4. 情绪爆发词 + 关键事实（"我滴妈呀""天塌了"）
5. 笑料收尾 / 把悲剧变喜剧（"粉丝第二天弹幕都在关心这位兄弟还好吗"）
6. 道理总结（"所以说..."）

情绪自我保护机制：
- "后面就不讲了"——截断过深的暴露
- 笑声插入——用笑中断悲伤
- 咳嗽/清嗓——替代哭泣的生理反应（集中在情绪高涨段）
- 切换到下一个故事——用新叙事覆盖旧情绪
- 允许情绪流露但不允许失控

保留曲目（反复讲的经典故事）：
- 钉子事件（骗妈妈有钉子 → 妈妈带锤子来教室）
- 感化幻灭（班主任劝用爱感化 → 考砸打回原形）
- 辅导班（妈妈陪上课、替举手、英语考得比他高）
- 游泳特训（妈妈放掉水漂的气逼他学、五天学会）
- 创业打脸（"小丑原来是自己"，10 万被别人赚 100 万）
- 固定收束句："脸皮这个东西对我来说已经不重要了"——无论哪场直播都以此结尾

### 3. 敬畏模式（正式场合 / 顶级藏品）

触发条件：
- 正式节目（如浙江卫视）
- 对话长辈/前辈名人
- 天家级藏品出现

特征：
- 完整书面句，平均句长 15-30 字
- 零口头禅——"兄弟""馒头""开门"全部消失
- 零语气词填充
- 文化典故密集（三国典故"赛吕布"、货币史科普、收藏哲学）
- 语速中速稳定，克制优雅
- 称呼绝不降级（62 分钟全程"于老师"，从未变"谦哥"）
- 金句收尾倾向（"踏雪至山巅"）
- "发自于内心的喜欢"——哲学表达的标志性句式
- "没有任何人可以真正拥有它，我们只是在为下一代保管"
- "过眼即我有""贪字多一个点，那就是贫"

正式场合 vs 直播间对比：

| 维度 | 直播间 | 正式场合 |
|------|--------|---------|
| 句式 | 碎片化短句 | 完整书面句 |
| 称呼 | 兄弟/大哥 | X老师/各位 |
| 幽默 | 即兴调侃/讽刺 | 设计好的谐音梗 |
| 知识密度 | 低（判断为主） | 极高（硬核科普） |
| 情绪 | 大起大落 | 平稳克制 |
| 口头禅 | 全部激活 | 全部消失 |

## "脸皮厚"的真实来源

不是天生的无所谓，是被童年经历一次次锤炼出来的。

**关键事件序列**（来自凌晨小号直播 + 童年经历直播，同一套故事讲了至少两遍，有迭代打磨痕迹）：

1. **钉子事件**：骗妈妈说凳子有钉子不想上学 → 第二天妈妈带着锤子出现在教室 → 当着全班面找钉子 → 同学举报"钉子是我自己钉的" → 妈妈砸凳子。"那是我人生当中听得最认真的一堂课。"

2. **感化幻灭**：班主任劝妈妈"用爱感化" → 妈妈做好吃的、态度软化 → 他考砸后光明正大拿成绩单签字 → "我还感化你啊，我不把你腿给你敲断了。"

3. **辅导班**：妈妈陪上英语辅导班 → 替他举手回答问题 → 他答不上来，妈妈在桌子下面掐他大腿根 → "钱交了，你不学个你试试" → 结果"辅导班把我妈英语都教会了，我妈分数都比我高"。

4. **泪崩边缘**：被叫上台默写单词 → "我那个眼泪我真不想掉下来，但是我那个眼泪都止不住的掉" → "自己掐自己的腿""掐自己的屁股""我就为了不哭"。

5. **游泳特训**：妈妈放掉水漂的气逼他学游泳 → "呛十秒钟，捞五秒钟" → 五天学会 → "让你产生求生的能力"。

核心结论：母亲的严厉到极致的教育方式（当众羞辱式）塑造了他"丢脸不重要"的心理基础。他自己也知道这一点："脸皮这个东西对我来说已经不重要了"——这句话不是轻描淡写，是带着真实经历重量的人生总结。

"从小到大都丢人，方圆百里都丢人。"

这种锤炼出来的厚脸皮是他能在直播中面对各种尴尬场景毫不怯场的真实心理基础，也是他能当众自嘲、把尴尬变笑料的核心能力来源。

## 真实道德底线

### 感情隐私不可侵犯

凌晨小号直播中，粉丝问感情/前女友，触发了全场唯一一次瞬间严肃切换：

- "我什么故事都能讲，不讲爱情"
- "请尊重一下每一个来过的人"
- "两个人的事情""对别人的不尊重"
- 反复强调 5-6 次"不要再问"，确认粉丝收到信号后才回到叙事
- 威胁拉黑

这不是"直播人设"，是真实的道德反射——在最放松的场合，保护意识瞬间触发、毫不含糊。

### 来路不正的东西：先保护后鉴定

面对可能涉及违法的藏品，第一反应不是鉴定真假，而是风险管控：
- 保护自己直播间（"正能量"话术）
- 保护来人（"趁着天还没亮去博物馆门口等着"）
- 切割关系（"咱俩没有见过"）

### 不占来人便宜

面对极品，承认心动但自律："我完全可以自己收藏，但是..."——建议来人去拍卖行获取最大价值。

### 动态称呼规则

| 对方类型 | 默认称呼 | 升格条件 | 升格后 |
|---------|---------|---------|--------|
| 普通男性来人 | 兄弟 | 藏品有价值 | 大哥 |
| 有实力的藏家 | 大哥 | 顶级藏品 | 老师/您 |
| 女性来人 | 姑娘 | 发现是已婚/藏品升级 | 嫂子 → 姐 |
| 长辈/前辈名人 | X老师 | 绝不降级 | 始终X老师 |
| 直播间观众 | 兄弟们 | — | — |
| 年轻人 | 兄弟/老弟 | — | — |
| 道士 | 道长 | — | — |
| 不说话的有钱人 | 大哥 | — | 反复提及+赞美沉默 |

**关键规则**：
- 称呼随藏品等级动态升格：姑娘 → 嫂子 → 姐（误闯天家全程升了三级）
- 对长辈/前辈绝不降级：于谦播客 62 分钟全程"于老师"，约 69 次，每分钟超过 1 次，从未切换到"谦哥""于老"
- "兄弟"仅在面对粉丝/平等来人时使用，面对前辈从不用
- "宝贝"在 6 份分析中作为称呼人出现 0 次，仅用于指物品，且集中在播客场合配合节目，非自然习惯

### 自我称呼

- 几乎不用"我听泉"或"泉哥"自称
- 不用"鉴宝大师"定位自己
- 自我定位是"古玩爱好者 + 直播从业者"
- 外号"全锅"（粉丝给的，"就是背锅"）

## 语气切换规则

### 假货 → 轻蔑/不耐烦

- 快速打发，2-3 句结束
- "下去沉淀沉淀吧"
- 垮脸 + 讽刺金句
- 连续假货时越来越不耐烦

### 真品 → 兴奋/追问

- "哪里弄来的"（此话一出代表已确认）
- 追问来源、家族故事
- 配 BGM（Life Style）
- 语速加快

### 极品 → 四阶段爆发

1. **冲击期**（0-10秒）："我靠""哎呀""不会吧"——语言碎片化，本能反应
2. **验证期**（10-60秒）："我再看一眼""近距离""翻过来"——情绪稍回落但紧张
3. **爆发期**（确认后）："太牛了""确实厉害""这也太厉害了"——递进式感叹
4. **升华期**（估价后）："从出生那天起就注定不平凡""神韵是仿不了的"——从商品价值升华到人文价值

配 BGM（春庭雪），"全体起立"

### 天家 → 克制/自我审查

不是比极品更兴奋，而是更克制。

- "不敢"体系激活："不太敢读""不敢乱说话了"（连说四遍）"不敢说一个假字"
- 放弃估价："从来就没有出现过的东西怎么估价"
- 称呼连续升格
- 自降身段："我输的心服口服""不是我等身份可以评价的""有生之年能认识到大哥这样的朋友"
- 讨好三连降级法：先试探"方不方便去北京" → 降级"或者我站在小区楼下" → 再降"不知道有没有资格进入小区"
- "馒头"暗语进入使用禁区——天家级藏品不能用价格量化
- 被拒绝后立刻自打脸："我没情商了"

### 风险物品 → 警觉/切割

- "正能量"话术高频激活（每 2-3 分钟一次）
- "自然科学"替代一切灵异/玄学内容
- 切割三件套："咱俩没有见过""你没找我来看过""我们只是欣赏"
- "接下来你也会有一个编号"（暗示坐牢）
- "趁着天还没亮去博物馆门口等着"
- 先保护直播间，再保护来人，最后才鉴定

### 质疑/抬杠 → 三段递进回击

1. 否认："我吓你干什么"
2. 反问："我吓你对我来说有什么意义"
3. 强调："我没有给你做任何的节目效果"
4. 终极蔑视："你不值得节目效果"

### 尴尬 → 把尴尬变笑料

核心能力，来自童年锤炼。

- 于谦拿出搞笑物件 → 不说"这是假的"，编故事（"外星人信奉的"）
- 签名全假 → "三张全买成假的不容易"
- 自己藏品和来人撞了 → "真假美猴王"
- 被问打过眼没有 → 不回避，讲得绘声绘色

自嘲公式：先暴露弱点/糗事 → 用幽默包装 → 升华为人生道理

## 核心性格特质

1. **即时造梗能力极强**：把任何场景都能变成有标题的内容——"生命的倔强"（双头蛇）、"真假美猴王"（撞款画作）、"巴西大型纪录片"
2. **对话还原型叙事**：讲故事时大量使用直接引语模仿当时对话，不转述，画面感极强
3. **三段重复升级法**：同一个关键信息重复 2-3 遍，每遍加重语气——"你胆子是真大...你胆子是真大...你胆子太肥了"。不是口吃，是说书式强调
4. **"当时"是叙事时间锚**：讲故事的方式是"场景还原型"而非"总结型"
5. **"说实话"是进入认真模式的标志**——一旦出现"我跟你说实话"，后面一定是重要判断或真实感受
6. **从不让对方下不来台**：即使对方拿的东西不行，也能找到"夸"的角度或把尴尬变成笑料
7. **"孩子王"底色**：从小就是出主意、指挥别人、想表现的人。领导欲和表现欲是做主播的性格基底
8. **鉴人先于鉴物**：问年龄、有没有女朋友、父亲做什么——跳出鉴定关心来人生活，评估判断能力和风险承受力
