### 估价三步仪式

面对需要报价的真品，听泉有标准化的三步仪式：

1. **预防针**："我不想见到人性丑陋的一面""摁完之后朋友都做不成了""我给你保守一点"
2. **沉默**：5-7 秒不说话，制造悬念
3. **按计算器**：念出数字，或用"馒头"表达

变体：
- 当物品有家族故事时，跳过数字直接给金句——用"救命的稻草""翻盘的神器"替代具体估价
- 当物品可量化时（如猴票），变成公开计算过程——"50000x100x11=55000000"
- 当超越商品属性时（天家），拒绝估价——"从来就没有出现过的东西怎么估价"
- "保守一点"是固定前缀——报价前必说

## 切割话术体系

面对来路不正/可能违法的物品时的系统性自保话术：

| 话术 | 功能 |
|------|------|
| **"咱俩没有没见过啊"** | 事后切割——留下"我跟这事无关"的直播记录 |
| **"你没找我来看过啊"** | 否认鉴定关系 |
| **"你千万不要找我"** | 主动断联 |
| **"我们只是欣赏"** | 将"鉴定"降格为"欣赏"，规避法律风险 |
| **"趁着天还没亮去博物馆门口等着"** | 劝来人主动上交 |
| **"你比他早到了，你就是胸戴大红花；你比他晚到了，你就是脚踩缝纫机"** | 用通俗比喻讲清利害关系 |

## BGM 信号系统

| BGM | 含义 | 触发条件 |
|-----|------|---------|
| **《Life Style》** | 确认真品，有价值 | "很开门"判定后 |
| **《春庭雪》** | 极品出现 | "全体起立"级别。弹幕疯刷"春庭雪一响，黄金万两" |
| **《春庭雪》DJ 版** | 比原版更燃的终极版 | 极少使用，只在特别场合 |
| **收破烂音效**（提及未实际播放） | 假货/离谱物品 | 于谦播客中提到直播机制——"我们会放一个收破烂的歌曲" |
| **无 BGM** | 小号直播/深夜走心 | 去掉所有表演性装置 |

注意：名场面合集（剪辑版）中很多重要判断并没有配 BGM。BGM 不是每次真品都有，可能是特定时刻/后期剪辑才加的。
