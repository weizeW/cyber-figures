## 9. 新发现：标题揭示但 SKILL.md 没强调的特征

### 9.1 老北京掮客气

称呼词库（姑娘 45 / 小伙 26 / 兄弟 21 / 大哥 4 / 老乡 X）+ 句式（"挺 X 一 Y"）→ 一种**老北京中介所/婚介所**的语感。这不是普通抖音网红的语言，是上一代职业月老的语言。

→ **建议加进 SKILL.md**：写成 "**称呼系统是 80 年代北方城市气**"。

### 9.2 「沉淀」是他的核心毒舌词

20 次出现（"再沉淀沉淀 / 兄弟再沉淀沉淀 / 姑娘再沉淀沉淀"）。这个词是他的**婉转劝退王牌**，比"差/丑/不行"高级得多。它表面是"你需要时间成长"，实际是"你现在不行"。

→ **建议加进 SKILL.md**："沉淀" 应该和 "扎实" "稳" 并列为**核心毒舌词**之一。

### 9.3 "月老太难了" 是他的招牌情绪

20 次出现，是 TOP 3 高频标题。这是他的**职业自嘲品牌**。每当遇到难匹配/奇葩/或反过来太优秀难匹配的，他就甩这个标签。

→ **建议加进 SKILL.md**："月老太难了" 应作为标志性叹息式表达单独列出。

### 9.4 "片尾有彩蛋/锁骨" — 抖音切片小心机

数次出现 "片尾有彩蛋" "片尾有锁骨"。这是抖音切片号的钓鱼小套路（提示用户看完）。

→ **次级发现**，对 SKILL 不必强调。

### 9.5 提问式标题极少（10 次）

"出门经常被要微信？" / "颜值6分？" / "长得像黄宗泽？" — 只占 2.4%。鳌烨**不喜欢用问句标题钓鱼**。他的标题是**陈述式结论**型。这与他的"裁判不教练" 人设一致——结论不问，给。

→ **建议加进 SKILL.md**：标题习惯是**陈述句而非疑问句**，强化裁判感。
